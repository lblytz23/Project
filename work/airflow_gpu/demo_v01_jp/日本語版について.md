## 💡 よくある質問

### Q: Python ファイル内のコメントが中国語だが問題ないか？

**A**: 問題ありません。コード自体は言語に依存せず、ドキュメントが日本語で書かれているため、使用に支障はありません。コードの動作は変わりません。

### Q: 中国語版と日本語版、どちらを使うべきか？

**A**: 
- 日本語が得意な方 → この日本語版をご使用ください
- 中国語が得意な方 → `demo_v01` ディレクトリの中国語版をご使用ください
- 機能は完全に同じです

### Q: コードのコメントを日本語に翻訳したい

**A**: 可能ですが、以下の点にご注意ください：
1. コードの動作は変わらないため、必須ではありません
2. 翻訳する場合は、コードの動作に影響しないよう注意してください
3. ドキュメントが充実しているため、コメントなしでも理解できるはずです

### Q: オリジナルの中国語版ドキュメントはどこにあるか？

**A**: 親ディレクトリの `demo_v01/` にあります：
```bash
cd ../demo_v01
ls *.md
```
